# NotifApp
Notif App is a simple Android application built to demonstrate basic Android development skills.  
The app features a single button that, when pressed, starts a **10-second timer**.  
A progress bar updates every second (in 10% increments), and a numeric value (10, 20, 30...100) appears above it.  

Once the progress reaches 100% (after 10 seconds), the app sends a **notification** with a title, message, and icon.
